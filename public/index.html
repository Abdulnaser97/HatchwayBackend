HatchWay API Proxy

Routes

  • http://localhost:8080/api/posts

    • Params

  • Tags (required) [ history, tech, health, ... ] | e.g. http://localhost:8080/api/posts?tags=history,tech
  • SortBy (optional) { id, reads, likes, poularity } | e.g. http://localhost:8080/api/posts?tags=history,tech&sortBy=popularity
  • Direction (optional) { asc, desc } | e.g. http://localhost:8080/api/posts?tags=history,tech&sortBy=popularity&direction=desc
